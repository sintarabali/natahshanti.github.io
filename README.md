# Natah Shanti Homestay Ubud (Website Scaffold)

This project is a modern, responsive hospitality website scaffold (HTML/CSS/JS) aligned with the provided brand identity and page requirements.

## Folder structure

- `index.html`, `rooms.html`, `facilities.html`, `gallery.html`, `explore-ubud.html`, `contact.html`
- `css/styles.css` (theme + layout)
- `js/main.js` (carousel, sliders, FAQ, booking modal)
- `assets/` (create image folders + icons as needed)

## WhatsApp phone

The site expects a WhatsApp phone number in the document root attribute:

- update in each HTML file: `data-whatsapp-phone="COUNTRYCODEPHONE"`

## Images

This scaffold uses placeholders under `assets/images/...`.
Replace them with your real images and update alt text as needed.
